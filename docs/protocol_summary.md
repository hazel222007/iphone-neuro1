# Protocol Summary

## Stimulus design
The exact blue room image was used as the base template. Monochromatic variants were created for the 17 CIELAB-derived colour conditions while preserving room geometry and luminance structure.

## Protocol modes
### 1. Static colour room
Best for strict physiological interpretation.

### 2. Subtle moving cue colour room
Best for older adults / dementia participants to reduce boredom and support sustained attention.

## Timing
- Initial neutral grey baseline: 90 s
- Neutral grey baseline before each colour: 30 s
- Colour-room exposure: 150 s
- Rest breaks: none

## Data export
Each protocol exports CSV and JSON event files.
The post-experiment questionnaire exports CSV and JSON rating files.
