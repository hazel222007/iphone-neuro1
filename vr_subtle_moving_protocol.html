NeuroColor Atlas - VR Subtle Moving Cue Colour Room Protocol

This VR-compatible protocol preserves the experiment flow and presents each existing stimulus image approximately 2 meters in front of the participant. It is intended for older adults/dementia participants to reduce boredom and maintain attention.

PhaseDuration
Initial neutral grey baseline90 seconds
Neutral grey baseline before each colour30 seconds
Colour-room exposure150 seconds
Rest breaksNone

On Android/compatible browsers this will attempt immersive WebXR. On iPhone Safari it automatically uses stereoscopic side-by-side Cardboard mode for the Irusu Monster headset. This layout is tuned for iPhone 16 Pro Max landscape viewing; if an iOS screenshot appears vertically stacked, it is the rotated screenshot representation. After pressing Start, the page attempts full-screen and landscape orientation.
